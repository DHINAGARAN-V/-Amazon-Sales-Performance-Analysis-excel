# Sales Performance Analysis Dashboard

## Project Overview

This project analyzes sales data using Microsoft Excel to understand revenue performance, product trends, and regional sales distribution. The dashboard provides insights into sales growth, product performance, and key business metrics to support data-driven decision-making.

## Objectives

* Analyze overall sales performance and revenue trends
* Identify top-performing products and categories
* Evaluate regional sales distribution
* Provide data-driven insights for business decisions

## Dashboard Features

* Total Sales, Profit, and Order KPIs
* Region-wise and category-wise sales analysis
* Monthly sales trend charts
* Product performance insights
* Pivot table summaries and interactive filters

## Tools Used

* Microsoft Excel
* Pivot Tables
* Excel Charts
* Data Analysis Techniques

## Key Insights

* Identification of high-performing products and regions
* Clear visualization of monthly sales trends
* Insights into category-level sales contribution
* Data-driven insights for business strategy
